HSS

HSS란 무엇인가?
Home Subscriber Server는 가입자 정보를 전체적으로 관리하는 서버이다.

HSS는 IDR을 다음 CASE에 C-SGN으로 전달한다. AccessRestrictionData가 설정되면 설정된 Access-Restriction-Data를 Subscription-Data AVP에 포함하여 전송한다. default Context가 변경되면 변경된 Default Context ID값이 반영된 APN-Configuration-Profile AVP를 Subscription-Data AVP에 포함하여 전송한다 새로운 Context ID가 추가되면 추가된 ID를 포함하여 전송한다 default Context가 변경되면서 기존 Context ID가 삭제되어도 IDR을 전송한다.

또한 DSR을 다음 CASE에 C-SGN으로 전달한다 ChargingCharacteristics가 0000으로 변경되면 DSR-Flags의 BIT2가 세팅된 상태로 DSR을 전송 한다. Default Ctxid 외에 다른 Ctxid 가입이 삭제되면 DSR-Flags의 BIT3이 세팅되어 DSR을 전송 한다. PdnCtx ID 8~16의 ContextID가 삭제 되는 경우에도 MME로 DSR을 전송하여야 한다.
